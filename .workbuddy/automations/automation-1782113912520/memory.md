**1. 概率表方向错位 ×3份**（严重 — 影响首页胜率显示）
- scotland-brazil.html：原「巴西胜 71% / 苏格兰胜 11%」→ 改为「苏格兰胜 11% / 巴西胜 71%」
- czech-mexico.html：原「墨西哥胜 65% / 捷克胜 13%」→ 改为「捷克胜 13% / 墨西哥胜 65%」

- 根因：报告生成时把「胜方/热门方」放在第1列，但build_site.py按schedule.json顺序取值（第1列=team_a）
